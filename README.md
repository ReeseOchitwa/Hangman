<h1>Hangman</h1>

<h2>Description</h2>
This project is a hangman game coded in python. It has a randomized list of possible words. It takes place in the console. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>Pycharm</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
